## Mapping Rules

- Every module should have an explicit output artifact.
- Every module should list the source files used to build it.
- If a module merges multiple sources, list all of them.
- If one source feeds multiple modules, record that in both tables.
- If a module is original and not source-derived, write `original` in `Source Files`.
- Keep module codes stable once publication has started.
