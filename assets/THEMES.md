# Theme Presets (Reusable)

Use one preset as project default, but keep all presets available.

## Supported presets
- `light`
- `dark`
- `colorful`
- `high-contrast`
- `warm`

## Suggested CSS variable tokens
- `--bg`
- `--surface`
- `--text`
- `--accent`
- `--accent-strong`
- `--border`

## Example values

- `--accent: #0f6f8e`
- `--accent-strong: #0a5368`
- `--border: #d8e1ef`
